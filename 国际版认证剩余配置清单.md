### A. `GEMINI_API_KEY` 申请步骤（最简单）

1. 打开 <https://aistudio.google.com/app/apikey>
2. 用你的 Google 账号登录
3. 点击 **Create API key**
4. 复制生成的 key
5. 发我，或你自己填到：

```bash
GEMINI_API_KEY=你的key
```

说明：
- 这是目前最快的方案，足够我们先把 intl 文本记账和 Prediction 真正跑通。
- 如果后面你想统一走 GCP 项目，也可以再迁移，不影响当前开发推进。

---

### B. `GOOGLE_VISION_API_KEY` 申请步骤

1. 打开 <https://console.cloud.google.com/>
2. 新建或选择一个 GCP Project
3. 左上角搜索 **Vision API**，进入后点击 **Enable**
4. 再打开 **APIs & Services → Credentials**
5. 点击 **Create credentials → API key**
6. 复制这个 key
7. 发我，或你自己填到：

```bash
GOOGLE_VISION_API_KEY=你的key
```

建议限制：
- 应用限制先可暂时不加，先联调通
- API 限制建议至少勾上 **Cloud Vision API**

---

### C. `FINNHUB_API_KEY` 申请步骤

1. 打开 <https://finnhub.io/>
2. 注册 / 登录
3. 进入 dashboard
4. 找到你的 API key（sandbox / free key 也可以先用）
5. 发我，或你自己填到：

```bash
FINNHUB_API_KEY=你的key
```

说明：
- 先用免费 key 就够我们做美股搜索和行情接入验证。
- 等需要更高频或更稳定额度时，再升级套餐。

### 1. Google OAuth
需要在 Google Cloud Console 创建并提供：
- iOS Client ID
- iOS Reversed Client ID
- Web / Server Client ID
- Android OAuth 配置（包名 + SHA1）

当前代码里已预留：
- `.env` -> `GOOGLE_IOS_CLIENT_ID`
- `.env` -> `GOOGLE_SERVER_CLIENT_ID`

建议再补：

### 2. iOS Google URL Scheme
拿到 `GOOGLE_IOS_REVERSED_CLIENT_ID` 后，需要加到：
- `ios/Runner/Info.plist`
- `CFBundleURLTypes`
- `CFBundleURLSchemes`

不加这一步，Google iOS 登录通常无法正常回跳。

当前状态：这部分已经配通并验证过，可暂时不用你再补。
