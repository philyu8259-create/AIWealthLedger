# App Icon Visual Dictionary

适用项目：AI Wealth Ledger / AI 记账鸭

## 目标

让 App Icon 与当前应用内视觉语言保持一致：现代、科技、清爽、带轻微毛玻璃与弥散高光质感，避免旧版纯色块或扁平图标造成割裂。

## 1. 核心底板：深邃空间渐变

请使用与首页“本月收支”主卡片一致的品牌渐变：

- Top-Left: `#6B4DFF`
- Bottom-Right: `#4A47D8`

补充要求：

- 不要使用单一纯色底。
- 在渐变底板上覆盖一层极其细微的 Noise，透明度约 `3%`。
- Noise 只用于增加实体感，不能做成明显颗粒。

## 2. 主体元素：3D 磨砂玻璃

主体可以是账本、鸭子、字母 A，或与“智能记账”直接相关的图形，但必须遵循以下原则：

- 主色使用纯白。
- 材质为 `Frosted Glass`（磨砂玻璃），不要使用扁平矢量风格。
- 边缘增加轻微白色高光，模拟悬浮在蓝紫底板上的折射感。
- 体积感要柔和，不要做成硬塑料或金属块。

## 3. AI 点睛元素

请将代码中的 `AiSparklesIcon` 作为中心或近中心点睛元素做 3D 化处理：

- 造型方向保持“三颗四角星（大星 + 两颗小星）”
- 主体可为白色或浅金白色
- 周围带一圈极轻的 Glow
- 作用是让用户第一眼就知道这是 AI 驱动的现代工具

## 4. 点缀色：呼应全局分类系统

为了呼应应用内分类图标的多彩数据感，请在主体边缘或背景远角加入极少量辅助光晕：

- 洋红色：`#E91E63`
- 翡翠绿：`#4CAF50`

使用规则：

- 只做“隐约透出”的 Glow，不做大片纯色块。
- 不能抢走蓝紫主底和白色主体的视觉中心。
- 两种辅助色总占比应非常低。

## 5. 构图与裁切

- 不要在 Icon 内再额外绘制一个边框。
- iOS / Android 会自动做圆角裁切，主体内容应饱满但不要贴边。
- 推荐主体略偏中上，保留下方视觉呼吸空间。

## 6. 禁止项

以下方向不再采用：

- 扁平纯色钱包 / 扁平鸭子
- 旧式拟物金属质感
- 大面积黑灰底
- 额外内框或按钮边框
- 与 App 内蓝紫品牌色无关的综合色调

## 7. 交付建议

建议设计师至少输出以下 3 个方向用于评审：

1. `Glass Duck + AI Sparkles`
2. `Glass Ledger + AI Sparkles`
3. `Glass Letter A + AI Sparkles`

每个方向至少提供：

- 1024x1024 主图
- 去背景预览图
- 深色 / 浅色桌面对比预览
- 是否含 alpha 的说明（iOS App Store 图标最终版本必须无 alpha）

## 8. 技术提醒

用于 iOS `AppIcon.appiconset/Icon-App-1024x1024@1x.png` 的最终文件必须：

- `RGB`
- `alpha = none`
- 无透明通道

否则会触发 App Store 校验错误：
`Invalid large app icon ... can’t be transparent or contain an alpha channel`
