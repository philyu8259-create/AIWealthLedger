# App Icon Delivery Checklist

## 必交文件

- [ ] 1024x1024 主图（最终候选）
- [ ] 1024x1024 无标注展示图
- [ ] 深色桌面预览图
- [ ] 浅色桌面预览图
- [ ] 设计说明图（标注材质、渐变、光晕）
- [ ] 备选方向至少 3 款

## 视觉检查

- [ ] 背景为品牌渐变 `#6B4DFF -> #4A47D8`
- [ ] 背景噪点极弱，不脏
- [ ] 主体为白色磨砂玻璃 / 软玻璃质感
- [ ] 主体轮廓在小尺寸下仍清晰
- [ ] 存在 AI Sparkles 点睛元素
- [ ] 辅助色仅作为弱光晕出现
- [ ] 无额外内边框
- [ ] 没有廉价塑料感或过度卡通感

## 技术检查

- [ ] 最终 iOS 图标为 RGB
- [ ] 最终 iOS 图标无 alpha
- [ ] 最终 iOS 图标无透明通道
- [ ] 文件命名规范明确
- [ ] 导出源文件保留（Figma / PSD / SVG）

## 评审标准

优先保留满足以下 4 条最多的方案：

1. 一眼看出是高端 AI 工具
2. 小尺寸识别度高
3. 与 App 内蓝紫视觉一致
4. 同时具备“财务工具”与“智能感”
